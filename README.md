# game-store-manager

Management system for an online game store: the program should allow
allow users to place orders from different game stores, view the catalog of games and prices, add and delete games and stores, as well as keep statistics on orders and generate sales reports.  C++

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/game-store-manager.git
cd game-store-manager
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
